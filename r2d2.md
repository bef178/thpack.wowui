# r2d2 for Vanilla (1.12)

「简单」插件合集。轻量，快速，无配置。

应用于[龟龟服](https://turtlecraft.gg/)，英文，1.12。

## 功能

功能较多，详见 `src/lua/wow/` 目录。大部分功能都可以顾名思义。部分解释如下：

- pixel-prefect
    - 自动调整UI缩放，使得像素能锐利对齐
    - 不作用于2560x1440分辨率
- preset-shortcuts
    - 个人使用的快捷键，输入命令后覆盖当前配置。`preset-macros` 类似
- open-all-bags
    - 单击背包时打开/关闭所有包
- flight-poi
    - 大地图上显示鸟点。待补全db
- blessingbar
    - 可对友方施放的buff技能，如 [力量祝福]
    - 按住 shift 使用这些技能的高阶版本，如 [强效力量祝福]
    - 亦含无需目标的短效 buff 技能，如 [正义圣印]
- aurabar
    - 无需目标的长效 buff 技能，如 [正义之怒]
- pda
    - 实时推荐当前「最合适」的技能，点击就放。目前仅配置防骑单刷A怪

## 安装

```
make
```

然后在 `target/` 目录下寻找 zip 文件。
